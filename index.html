Com este propósito desenvolvemos uma ferramenta com o intuito de ser acessível, eficaz e de fácil uso contra um dos tipos mais comuns de ciberataques, o phishing.   • O que é Phising?

    Phishing é um trocadilho que refere-se à palavra "fishing" do inglês, significando "pescaria", porém nesta você é a pesca! Trata-se de uma técnica de fraude cibernética na qual os criminosos tentam enganar as pessoas, fazendo-se passar por fontes confiáveis. As principais características do phishing incluem a criação de mensagens convincentes que geralmente envolvem um senso de urgência, ameaças ou incentivos para que as vítimas ajam rapidamente. Os links geralmente direcionam para sites falsos que se assemelham aos originais, mas são projetados para coletar dados pessoais. Dentre as diversas técnicas fraudulentas nosso foco está em identificar os links de redirecionamento para o roubo de dados antes que você seja pescado!

  • Extensão para navegadores

    Nossa extensão TideWave Blocker funciona da seguinte forma: a partir do momento em que está instalada, toda página web que termina de ser carregada passa por uma verificação minuciosa do conteúdo procurando links de redirecionamento, cada link identificado é enviado para uma base de dados que armazena links conhecidos por serem malicioso. Desta maneira, retornamos o resultado da análise informando se o site é seguro e se possui ou não links maliciosos, identificando-os de forma explicita a você!
